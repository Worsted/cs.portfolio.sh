HTML

Background: HTML is an acronym that stands for Hypertext Markup Language. HTML is used as a coding language for text files to change font, color, and other effects on websites. HTML is able to incorporate multiple objects onto a webpage but no real way to deal with the actual organization of the objects HTML is used for the basic design of websites. HTML originated from a contractor of CERN, that wanted a text software for CERN researchers to use and share documents HTML is written as a markup language. The current version of HTML that we are using today is HTML5. HTML didn't start out as HTML5, programmers have been updating HTML constantly and releasing new versions of the language. HTML is used by browsers to interpret the content of the page, not represent the actual HTML tags and coding behind it. HTML is able to embed scripts from a language called Javascript which affect the behavior of HTML webpages. Tim-Berners Lee is the inventor of HTML.

CSS

Background: CSS is an acronym that stands for Cascading Style Sheet. CSS is used as a coding language for web pages to keep information in certain places and controls most of a sites' aesthetics only for sites written in a markup language. CSS is most often used for sites written in HTML and XHTML. CSS is mainly used from separating content from presentation. CSS was written by W3C which is the main organization responsible for designing World Wide Web. The main purpose of CSS was to allow the separation of documents and presentation on a web page. CSS is written to compliment web pages written in markup languages. CSS controls most of the visual aspects and ordering of objects on a web page. CSS has been updated to bring us the latest version, CSS 4, but each variation of CSS is used to fit the needs of which piece of technology it will be used in. CSS uses simplified words to depict various style properties. Hakon Wium Lie is the inventor of CSS.

Similarities

  • HTML and CSS were designed to collaborate with each other for webpage designs
  • Both languages are for designing web pages
  • CSS fixed a lot of problems that HTML had with the placement of objects in a webpage.
  • HTML and CSS were made by the same organization
  • HTML is used for the primary makeup of a webpage, while CSS is able to go the extra mile to make that website just a bit more neat and colorful
  • HTML is used more than CSS, but most well done HTML written webpages have CSS in it as well.
  • The maker of CSS had worked with the maker of HTML before.
